### code style
- use python 3
- use sync python, not asyncio
- use camelCase, not snake_case, for all variable and function names
- only add comments where the code is otherwise confusing. let the clear code speak for itself, sans comments
- use single quotes for strings everywhere possible, eg 'string' over "string"
- dont include any type declarations
- line wrap code lines at 79 characters per line
- line wrap docstrings and comments 72 characters per line
